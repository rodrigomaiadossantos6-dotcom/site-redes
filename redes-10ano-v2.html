Cada computador numa rede tem uma morada única — o endereço IP.

É como o número da tua casa! Sem morada, ninguém sabe onde entregar a carta. 📬

Clica no computador e depois no IP que queres atribuir!

Passo 3 de 5
Além do IP, cada computador precisa saber qual é a sua "vizinhança" — quais os computadores da mesma rede.

Isso chama-se máscara de sub-rede. Para redes normais usa-se sempre o mesmo valor!
🏘️
Analogia
O IP é a morada. A máscara é o código postal — define o bairro!
🎯 Qual é a máscara mais usada nas redes de casa/escola?
